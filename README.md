# gui-practice-nodejs
Practice of nodejs

## preparation

~~~
apt install make gpg
make prepare-nodejs
make prepare-ejs
~~~

## 1st step: Hello, World applcation

 1. Invoke hello world application
    ~~~
    node app0.js
    ~~~
 1. Invoke browser
    ~~~
    firefox http://localhost:3000/
    ~~~

## 2nd step: Use EJS

 1. Preparation
    ~~~
    npm install ejs
    ~~~
 1. Execution
    ~~~
    node app.js
    ~~~
 1. Invoke browser
    ~~~
    firefox http://localhost:1234/
    ~~~
